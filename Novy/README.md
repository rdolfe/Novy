# Novy Project

## Overview

Novy is a full-stack application that combines a backend built with Node.js and Express, and a frontend developed using React. The application is designed to provide a seamless user experience with efficient data handling and a responsive interface.

## Features

- **Backend**: 
  - RESTful API built with Express.
  - Database integration for data persistence.
  - Environment variable management for sensitive information.

- **Frontend**: 
  - React-based user interface.
  - Component-based architecture for reusability.
  - Vite as the build tool for fast development.

## Installation

Follow the instructions in the `INSTALLATION.txt` file to set up the project on your local machine.

## Usage

1. Start the backend server by navigating to the `backend` directory and running `node server.js`.
2. Start the frontend application by navigating to the `frontend` directory and running `npm run dev`.
3. Access the application in your browser at `http://localhost:5174`.
